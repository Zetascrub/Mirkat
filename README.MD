# M.I.R.K.A.T - Making Initial Recon: Keen Assessment Tool

Description

M.I.R.K.A.T (Making Initial Recon: Keen Assessment Tool) is a Python-based automation tool designed to streamline and enhance the process of penetration testing. It integrates various scanning and assessment tools, providing a unified interface to conduct comprehensive security audits efficiently.
Installation

To install M.I.R.K.A.T, clone the repository and set up the required dependencies:

bash

    git clone https://github.com/zetascrub/mirkat.git
    cd mirkat
    pip install -r requirements.txt
